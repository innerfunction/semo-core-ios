IFMessageTargetContainer Protocol Reference

Conforms to NSObject
Declared in IFMessageTargetContainer.h

Overview

Protocol for dispatching messages to named targets. Components which contain addressable message targets should implement this protocol.

– dispatchMessage:sender: required method

Try dispatching the message to message targets within the current compnent.

- (BOOL)dispatchMessage:(IFMessage *)message sender:(id)sender

Parameters

message

The message.
sender

The component sending the message.

Return Value

Returns true if the message was dipatched (received and acted upon) by some target.

Discussion
