IFTypeInfo Class Reference

Inherits from NSObject
Declared in IFTypeInfo.h
IFTypeInfo.m

Overview

A class for inspecting all of the declared properties on an object. Generates a list of all the publically accessible properties on an object’s class and all of its superclasses. Information can then be accessed for each property.
