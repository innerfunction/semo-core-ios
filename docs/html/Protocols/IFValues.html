IFValues Protocol Reference

Conforms to NSObject
Declared in IFValues.h

Overview

A protocol for providing access to different representations of an underlaying value. Values are referenced by key path.

– getValue: required method

Return the bare value without any type conversion.

- (id)getValue:(NSString *)keyPath

Discussion

Return the bare value without any type conversion.

– getValueAsLocalizedString: required method

Get a value as a localized string. The underlying string value should correspond to a resource ID in strings.xml, and the result is returned for the current platform locale. Returns the keyPath argument if the value can’t be found.

- (NSString *)getValueAsLocalizedString:(NSString *)keyPath

Discussion

Get a value as a localized string. The underlying string value should correspond to a resource ID in strings.xml, and the result is returned for the current platform locale. Returns the keyPath argument if the value can’t be found.
